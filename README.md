# DeVaughn Clark - Speaker Website

A modern, premium speaker portfolio website built with Next.js, Tailwind CSS, and TypeScript.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Add your images to `/public/images/` with these exact names:
   - `hero.jpg` - The main hero image (DeVaughn on stage with arm raised - IMG_9322.jpg)
   - `headshot.jpg` - Professional headshot against brick wall (DSC00027.JPEG)
   - `speaking-1.jpg` - Speaking photo 1 (IMG_9322.jpg or IMG_9323.jpg)
   - `speaking-2.jpg` - Speaking photo 2 (IMG_9323.jpg)
   - `podcast-1.jpg` - Podcast photo (A7404666.jpeg)
   - `podcast-2.jpg` - Podcast photo with Ciara (A7404798.jpeg)
   - `lifestyle.jpg` - Lifestyle/seated photo (IMG_9216.jpeg)

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000)
